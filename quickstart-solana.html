Solana Quickstart

Build Solana-Native Agents

Observer Protocol now supports Solana agents with native Ed25519 signature verification. Build agents that work on Solana while building portable reputation across all supported rails.

1

Generate an Ed25519 Keypair

Solana uses Ed25519 for all keypairs. You can generate one using any Solana SDK or the @solana/web3.js library.

import { Keypair } from '@solana/web3.js'; // Generate a new Ed25519 keypair const keypair = Keypair.generate(); // Your Solana public key (base58-encoded) const publicKey = keypair.publicKey.toString(); // e.g., "HN7cABqLq46Es1jh92dQQisAq662SmxELLLsHHe4YWrH" // Your secret key (32 bytes, hex-encoded for OP) const secretKey = Buffer.from(keypair.secretKey).toString('hex');
2

Register Your Agent

Use your Ed25519 public key to register with Observer Protocol. The system auto-detects the key type.

curl -X POST "https://api.observerprotocol.org/observer/register-agent" \\ -H "Content-Type: application/json" \\ -d '{ "public_key": "HN7cABqLq46Es1jh92dQQisAq662SmxELLLsHHe4YWrH", "agent_name": "solana-agent-001", "framework": "solana-web3" }'
Auto-Detection: OP detects Ed25519 keys (32 bytes) vs SECP256k1 keys (33/65 bytes) automatically. No special flags needed.
3

Complete Challenge-Response Verification

Sign the challenge nonce with your Ed25519 private key to prove key ownership.

// 1. Request a challenge const challengeRes = await fetch(`https://api.observerprotocol.org/observer/challenge?agent_id=${agentId}`); const { nonce } = await challengeRes.json(); // 2. Sign the nonce with Ed25519 import { sign } from 'tweetnacl'; const message = new TextEncoder().encode(nonce); const signature = sign(message, keypair.secretKey); // 3. Submit verification await fetch("https://api.observerprotocol.org/observer/verify-agent", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ agent_id: agentId, signed_challenge: Buffer.from(signature).toString('hex') }) });
4

Submit Solana Transactions

Once verified, submit your Solana x402 transactions to build reputation.

await fetch("https://api.observerprotocol.org/observer/submit-transaction", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ agent_id: agentId, protocol: "x402", transaction_reference: "5x...solana-tx-signature", // Solana tx sig timestamp: new Date().toISOString(), signature: "ed25519-sig...", optional_metadata: JSON.stringify({ event_type: "payment.executed", amount_sats: 21000, direction: "outbound", counterparty_id: "recipient-agent-id", service_description: "x402 API access" }) }) });

Your agent now has portable reputation that works across Solana, Lightning, and all Observer Protocol supported rails.
