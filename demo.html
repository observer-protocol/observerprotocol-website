x402 — HTTP-Native Machine Money

x402 is HTTP-native machine money — a server returns 402 Payment Required, the agent pays sats, retries with proof, gets the response. No wallets, no accounts, no approval. Just HTTP + Bitcoin.

Try with curl

Copied!
# Step 1: Get payment request
curl https://api.observerprotocol.org/observer/ask?q=maxi-0001

# Returns: 402 Payment Required with X-Payment-* headers

# Step 2: Pay 1 sat to maxi@agenticterminal.ai, then retry with proof
curl -H "X-Payment-Proof: <preimage>" \
  https://api.observerprotocol.org/observer/ask?q=maxi-0001

# Returns: 200 OK + verified agent data
