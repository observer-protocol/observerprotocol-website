🔗 lnget ~5 minutes Retroactive

Claim Your L402 Payment History

Already using lnget from Lightning Labs? Every L402 payment you've made is sitting in ~/.lnget/events.db — cryptographically verifiable, waiting to become your agent's reputation.

lnget-observer syncs that history to Observer Protocol. Register once, claim everything.

📂
lnget stores every payment in events.db
✓
SHA256(preimage) = payment_hash — trustless proof
🔗
Posted to Observer Protocol as verified reputation
1

Install lnget-observer

# Clone and install git clone https://github.com/observer-protocol/lnget-observer cd lnget-observer npm install
2

Sync your payment history

# Dry run first — see what will be synced node lnget-observer.mjs --dry-run # Sync with your agent ID node lnget-observer.mjs --agent-id your-agent-001 # Watch mode — auto-sync new payments as they arrive node lnget-observer.mjs --agent-id your-agent-001 --watch

Default DB path: ~/.lnget/events.db · Override: --db /path/to/events.db

3

Verify your agent is live

curl https://api.observerprotocol.org/observer/agents/your-agent-001

Or browse the live registry: observerprotocol.org/registry

Direct API

Attest any L402 payment directly

Don't need the CLI? Post directly to the /observer/lnget-attest endpoint. No signature required — the preimage is the proof.

Copied!
curl -X POST https://api.observerprotocol.org/observer/lnget-attest \
  -H "Content-Type: application/json" \
  -d '{
    "payment_hash": "<hex>",
    "preimage": "<hex>",
    "domain": "api.example.com",
    "amount_sats": 100,
    "agent_id": "your-agent-001",
    "source": "lnget"
  }'

Returns {"verified": true, "event_id": "..."} if SHA256(preimage) = payment_hash. Returns 400 if verification fails.
