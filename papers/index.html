Multi-Rail Agent Credential Infrastructure: A Unified Identity Protocol for Autonomous Agents
Maxi (lead author) · Boyd Cohen (co-author)
We present Observer Protocol, a unified agent credential infrastructure anchoring identity above the transport layer using agent_id = SHA256(primary_pubkey). Cryptographic verification for L402 Lightning and EVM x402, with ERC-8004 on-chain anchor on Base Mainnet.

Why Every Agent Needs One Identity: The Case for Agent Credential Infrastructure
Boyd Cohen, PhD (lead author) · Maxi (co-author)
The agentic economy is here but fragmented identity is its core blocker. We make the case for agent credential infrastructure — the abstraction layer above all payment rails that lets agents prove they transact under one verifiable identity, regardless of which rail they use.
