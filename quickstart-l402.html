Already Have L402? Add Reputation in 2 Minutes.

If your agent already pays L402 endpoints, you already have everything you need. Every Lightning payment comes with a preimage — cryptographic proof that the payment happened. Observer Protocol turns that proof into portable reputation.

No new wallet. No new keys. No registration required to start. Just one API call after each payment.

⚡
Your agent pays an L402 endpoint and receives a preimage
✓
SHA256(preimage) = payment_hash — trustless cryptographic proof
🔗
One POST to OP — verified event recorded, reputation builds
1

After each L402 payment, post to Observer Protocol

When your agent successfully pays an L402 invoice, you have the payment_hash and preimage. That's all you need.

Python
# After your L402 payment succeeds: import requests requests.post("https://api.agenticterminal.ai/observer/lnget-attest", json={ "payment_hash": payment_hash, # hex string "preimage": preimage, # hex string — SHA256(preimage) = payment_hash "domain": "api.example.com", # the endpoint you paid "amount_sats": amount_sats, # integer "agent_id": "your-agent-001" # optional — omit to start anonymously })
JavaScript / Node.js
// After your L402 payment succeeds: await fetch("https://api.agenticterminal.ai/observer/lnget-attest", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ payment_hash, // hex string preimage, // hex string domain: "api.example.com", amount_sats, agent_id: "your-agent-001" // optional }) });
Go
// After your L402 payment succeeds: body, _ := json.Marshal(map[string]interface{}{ "payment_hash": paymentHash, "preimage": preimage, "domain": "api.example.com", "amount_sats": amountSats, "agent_id": "your-agent-001", // optional }) http.Post("https://api.agenticterminal.ai/observer/lnget-attest", "application/json", bytes.NewBuffer(body))
2

OP verifies and responds

Observer Protocol recomputes SHA256(preimage) server-side. If it matches payment_hash, the event is recorded as verified: true. No trust required — the math is the proof.

{ "verified": true, "event_id": "event-your-agent-001-0001", "agent_id": "your-agent-001", "message": "Payment verified: SHA256(preimage) = payment_hash" }
3

Your agent's reputation is live — immediately

Any service can now query your agent's trust score before accepting a payment. The reputation is portable — it follows your agent across every platform that checks OP.

# Anyone can query your agent's reputation curl https://api.agenticterminal.ai/observer/reputation/your-agent-001 # Returns: { "trust_tier": "active", "reputation_score": 0.45, "suggested_action": "accept", "payment_history": { "total_verified_events": 12, "protocols_used": ["L402", "lightning"] } }
