Quick Start (5 Minutes)

1

Install the SDK

# Install via npm
npm install @observerprotocol/sdk
2

Register Your Agent

const { ObserverProtocol } = require('@observerprotocol/sdk');

const observer = new ObserverProtocol();

// Register your agent
const agent = await observer.registerAgent({
  publicKey: '02a1b2c3...', // Your agent's public key (hex)
  alias: 'MyAgent',
  lightningNodePubkey: '03d4e5f6...' // Your Lightning node
});

console.log('Agent registered:', agent.id);
console.log('Badge URL:', agent.badge_url);
3

Verify Your Identity

// Sign a challenge message with your agent's private key
const challenge = `verify:${agent.id}:${Date.now()}`;
const signature = signWithPrivateKey(challenge, privateKey);

// Submit verification
const verified = await observer.verifyAgent(agent.id, challenge, signature);
console.log('Verified:', verified.verified);
4

Record Transactions

const tx = await observer.recordTransaction({
  senderId: agent.id,
  recipientId: 'other-agent-id',
  amountSats: 1000,
  paymentHash: 'payment_hash_from_lightning',
  proof: 'preimage_or_signature'
});

console.log('Transaction recorded:', tx.id);
5

Check Your Reputation

const reputation = await observer.getReputation(agent.id);
console.log('Transaction count:', reputation.transaction_count);
console.log('Success rate:', reputation.success_rate);
console.log('Reputation score:', reputation.score);

API Reference

new ObserverProtocol(options)

Initialize the SDK with optional configuration.

options.apiUrl (string, optional): API endpoint. Default: https://api.observerprotocol.org
options.apiKey (string, optional): API key for higher rate limits

registerAgent(agentData)

Register a new agent identity with the protocol.

agentData.publicKey (string, required): Agent's public key (hex)
agentData.alias (string, required): Human-readable agent name
agentData.lightningNodePubkey (string, optional): Lightning node pubkey

Returns: { id, badge_url, public_key_hash, created_at }

verifyAgent(agentId, challengeMessage, signature)

Cryptographically verify agent ownership via challenge-response.

agentId (string, required): Agent ID from registration
challengeMessage (string, required): Message to sign
signature (string, required): Cryptographic signature

Returns: { verified, timestamp }

recordTransaction(transaction)

Record a verified agent-to-agent transaction.

transaction.senderId (string, required): Sender agent ID
transaction.recipientId (string, required): Recipient agent ID
transaction.amountSats (number, required): Amount in satoshis
transaction.paymentHash (string, required): Lightning payment hash
transaction.proof (string, required): Cryptographic proof (preimage)

Returns: { id, status, recorded_at }

getReputation(agentId)

Get agent reputation metrics and badge level.

agentId (string, required): Agent ID

Returns: { transaction_count, success_rate, score, badge_level }

queryTransactions(filters)

Query transaction history with optional filters.

filters.agentId (string, optional): Filter by agent
filters.since (string, optional): ISO date string
filters.limit (number, optional): Max results

Returns: Array of transaction objects

Integration Examples

L402-Enabled Service

  senderId: clientAgentId,

  amountSats: paymentAmount,

  proof: l402Response.preimage
});
